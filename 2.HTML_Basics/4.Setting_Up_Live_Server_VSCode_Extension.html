Setting_Up_Live_Server_VSCode_Extension

What is Live Server?

  • It's an extension allow you to open HTML web pages on a little development server

  • To deal with live server you need first to install it from "Vscode extension "

  • In order to open live server .. right click on you Html file .. choose open with live server .

  • Once it's open you'll notice that on the brower's address bar current path of your html file .

  • With live server you don need to reload your web page manulay .. because it's will reload automatically.

  • Notice : press ALT + L + O to open live server
