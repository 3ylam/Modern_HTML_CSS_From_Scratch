What is an Internet?

  • A global network of computers and other devices

  • Each device has it's own Ip address , which is uniquely identifies the device other the internet also used to tracing the device over the network

  • IP address assigned to your computer by server provider

  • Your router should have public IP address to be for internet to see it .

  • Usually it comes in the form of four numbers , there is version4 and 6 .. but version 6 is more longer

  • There are a lot of websites can tell you about your IP .. but the most famous one is WhatIsmyIpAddress

    • In case of multiple devices belong to your office or home network , each of this devises should have what called private IP address

  • Computers talk to each other via TCP/IP protocol

    • A protocol is just a way for computers to communicate with each other just like our natural language.

  • To more info you can getting into computer network to discover more stuff such as [different network layer , how data packets are sent , ] but that is not important for web development.

  • The most important protocol for us as a web devs is HTTP protocol

    • HTTP stands for Hyper Text Transfer Protocol

    • Definitely you notice that when you typing something on address bar of your browser

    • HTTP one of the many app protocols .. it is called an application layer protocol "actually run over the tcp/ip model"

    • Also we have SMTP "deal with email , FTP deal with transferring files over the internet"

    • HTTPresponsible for handling request and respond between web browser and server side
